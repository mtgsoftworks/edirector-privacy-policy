# eDirector™ Football Manager 26 - Privacy Policy Page

This is a simple static HTML page hosting the privacy policy for eDirector™ Football Manager 26.

## Hosting

This page is hosted on Netlify.
URL: <https://edirector-privacy.netlify.app/privacy-policy.html>

## Content

- **privacy-policy.html**: The main privacy policy document complying with App Store and Play Store requirements, including Google AdMob usage disclosures.
- **app-ads.txt**: Authorization file for AdMob.

## Updating

1. Edit `privacy-policy.html`
2. Commit and push changes
3. Netlify will automatically deploy
